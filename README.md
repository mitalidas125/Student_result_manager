# 🧑‍🎓 Student Result Manager

A simple Python GUI application to store and manage student results using a CSV file.

---

## ✨ Features

- Add student name, roll number, and subject marks
- Automatically calculates total, average, and grade
- Stores data in `student_data.csv`
- Easy-to-use GUI built with `tkinter`
- Clean and minimal design
